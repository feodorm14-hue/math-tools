# files.md — что будет в проекте

Список модулей, формул и калькуляторов. Бэкенд: Python + FastAPI.
Уровень: 5–7 класс.


## Калькуляторы

дроби
сложение, вычитание, умножение и деление дробей
принимает числитель и знаменатель двух дробей, возвращает результат и сокращённую форму
пример: 2/3 + 3/4 = 17/12

проценты
найти процент от числа, найти сколько процентов одно число составляет от другого
пример: сколько процентов 45 от 180? → 25%

уравнения первой степени
решает уравнение вида ax + b = c
принимает a, b, c — возвращает x
пример: 2x + 3 = 11 → x = 4

пропорции
принимает три из четырёх значений a/b = c/d — находит неизвестное
пример: 4/6 = x/9 → x = 6

периметр и площадь
квадрат, прямоугольник, треугольник, круг, трапеция, параллелограмм
принимает нужные стороны или радиус — возвращает P и S

объём фигур
куб, прямоугольный параллелепипед, цилиндр, конус, пирамида
принимает размеры — возвращает объём и площадь поверхности

теорема Пифагора
принимает два катета — возвращает гипотенузу
или принимает гипотенузу и один катет — возвращает второй катет
пример: a=3, b=4 → c=5

степени и корни
вычисляет a^n, квадратный корень, кубический корень
пример: 2^10 = 1024, sqrt(144) = 12

НОД и НОК
наибольший общий делитель и наименьшее общее кратное двух или трёх чисел
пример: НОД(48, 36) = 12, НОК(4, 6) = 12

статистика (базовая)
принимает список чисел
возвращает: среднее арифметическое, медиану, моду, размах


## Библиотека формул

каждая формула хранится в SQLite: название, LaTeX-строка, категория, описание

геометрия на плоскости
площадь квадрата: S = a²
площадь прямоугольника: S = a · b
площадь треугольника: S = (a · h) / 2
площадь круга: S = π · r²
длина окружности: C = 2 · π · r
площадь трапеции: S = (a + b) / 2 · h

пространственная геометрия
объём куба: V = a³
объём параллелепипеда: V = a · b · c
объём цилиндра: V = π · r² · h
объём конуса: V = (1/3) · π · r² · h
объём пирамиды: V = (1/3) · S · h

алгебра
формула дискриминанта: D = b² - 4ac
корни квадратного уравнения: x = (-b ± √D) / 2a
разность квадратов: a² - b² = (a - b)(a + b)
квадрат суммы: (a + b)² = a² + 2ab + b²
квадрат разности: (a - b)² = a² - 2ab + b²

арифметика
сумма n первых натуральных чисел: S = n(n+1) / 2
нахождение процента: P = (x / n) · 100
арифметическая прогрессия: aₙ = a₁ + (n-1) · d, S = n(a₁ + aₙ) / 2

физика (7 класс)
скорость: v = s / t
путь: s = v · t
давление: P = F / S
плотность: ρ = m / V
КПД: η = A_полезн / A_затрач · 100%

## Тесты (Vitest)

тесты живут рядом с модулями или в папке tests/

tests/arithmetic.test.ts
проверяет калькулятор дробей: 1/2 + 1/3 = 5/6
проверяет проценты: 25% от 200 = 50
проверяет НОД и НОК

tests/geometry.test.ts
площадь круга с r=5: ≈ 78.54
теорема Пифагора: 3, 4 → 5
объём куба со стороной 3: 27

tests/algebra.test.ts
уравнение 2x + 3 = 11 → x = 4
пропорция 4/6 = x/9 → x = 6

запуск тестов:
npx vitest         — следит за изменениями и перезапускает
npx vitest run     — один прогон

скрипты в package.json:
"dev":   "vite"           — запускает dev-сервер фронта
"build": "vite build"     — собирает фронт в dist/
"test":  "vitest run"     — запускает все тесты один раз
"watch": "vitest"         — режим слежения за файлами

## Docker

docker-compose.yml — поднимает оба сервиса одной командой

backend/Dockerfile
  образ: python:3.11-slim
  устанавливает requirements.txt
  запускает uvicorn на порту 8000 с флагом --reload

frontend/Dockerfile
  образ: node:20-alpine
  устанавливает npm-зависимости
  запускает vite dev-сервер на порту 5173

vite.config.ts — проксирует /api/* → http://backend:8000
  благодаря этому фронт общается с бэком по имени сервиса внутри Docker-сети

backend/.dockerignore — исключает __pycache__, .env, store.db
frontend/.dockerignore — исключает node_modules/, dist/, .env

команды:
docker compose up --build     — собрать и запустить всё
docker compose up -d          — запустить в фоне
docker compose down           — остановить
docker compose logs backend   — логи только бэкенда

после запуска:
  фронт:  http://localhost:5173
  бэкенд: http://localhost:8000
  документация API: http://localhost:8000/docs
