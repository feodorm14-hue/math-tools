Пропорция

Пропорция — равенство двух отношений: a/b = c/d. Это значит, что отношение первого к второму такое же, как третьего к четвёртому.

Основное свойство пропорции: произведения крест-накрест равны, то есть a×d = b×c. Из этого следует: d = b·c / a.
Пример: 2/3 = 4/? → d = 3×4/2 = 6.

a / b = c / d → найти d

Дроби

Дробь — запись вида a/b, где a — числитель, b — знаменатель (b ≠ 0). Дробь показывает, какая часть от целого взята.

Правила: при сложении/вычитании нужен общий знаменатель; при умножении — перемножить числители и знаменатели; при делении — умножить на перевёрнутую дробь. Калькулятор даёт точный результат без округлений. Поддерживаются +, −, ×, ÷ и скобки.

Проценты

Процент — сотая часть числа (от лат. «pro centum» — из ста). Знак %.

Формулы: X% от N = N × X / 100. Обратная задача — сколько процентов X составляет от N: P = X / N × 100.
Пример: 20% от 150 = 150×20/100 = 30. И наоборот: 30 — это 20% от 150.

НОД и НОК

НОД (наибольший общий делитель) — самое большое число, на которое делятся оба числа без остатка. Пример: НОД(12, 8) = 4, потому что 4 делит и 12, и 8, и больше такого числа нет.

НОК (наименьшее общее кратное) — самое маленькое число, которое делится на оба числа. Пример: НОК(4, 6) = 12, потому что 12 делится и на 4, и на 6. Связь: НОК(a,b) = a × b / НОД(a,b).

Степени и корни

Степень aⁿ — это произведение n одинаковых множителей a. Например, 2⁴ = 2×2×2×2 = 16.

Квадратный корень √a — число, которое при возведении в квадрат даёт a. Например, √25 = 5.
Кубический корень ∛a — число, куб которого равен a. Например, ∛27 = 3.
Степень и корень — обратные операции: √(a²) = a.

Статистика

Анализ набора чисел: среднее, медиана, мода, размах, минимум и максимум.

Медиана — среднее значение упорядоченного ряда. Числа сортируются по возрастанию, и берётся центральный элемент (или среднее двух центральных, если чисел чётное количество). Пример: медиана 2, 3, 5, 7, 8 — это 5.
Мода — наиболее часто встречающееся значение. Пример: в ряду 2, 3, 3, 5, 7 мода равна 3, потому что она встречается чаще всего.

Скорость · Время · Расстояние

Основная формула движения: v = s / t

Где v — скорость (км/ч), s — расстояние (км), t — время (ч). Из неё получаем: s = v × t и t = s / v.
Пример: велосипедист едет 15 км/ч, 2 часа. Путь = 15 × 2 = 30 км.

Делители числа

Делитель числа n — натуральное число, на которое n делится без остатка.

Простое число — делится только на 1 и на само себя (2, 3, 5, 7, 11…). Составное — имеет другие делители.
Пример: делители 12 → 1, 2, 3, 4, 6, 12 (составное). Делители 7 → 1, 7 (простое).
Алгоритм: проверяем все числа до √n — это быстрее, чем до n.

Факториал и комбинаторика

Факториал n! = 1 × 2 × 3 × … × n. Например, 5! = 120. По определению 0! = 1.

Сочетания C(n,k) — количество способов выбрать k предметов из n (порядок не важен). C(5,2) = 10.
Размещения A(n,k) — количество способов выбрать и расставить k предметов из n (порядок важен). A(5,2) = 20.
Формулы: C(n,k) = n!/(k!(n−k)!), A(n,k) = n!/(n−k)!

n! = 1 × 2 × 3 × … × n

C(n,k) = n! / (k! × (n−k)!)

A(n,k) = n! / (n−k)!

Уравнения

Уравнение — запись вида f(x) = g(x), где нужно найти значение x.

Линейное уравнение ax + b = 0 решается: x = −b / a. Например, 2x + 6 = 0 → x = −3.
Квадратное уравнение ax² + bx + c = 0 решается через дискриминант: D = b² − 4ac. Если D > 0 — два корня, D = 0 — один корень, D < 0 — нет вещественных корней. Корни: x = (−b ± √D) / 2a.

Координатная плоскость

Координатная плоскость — это две перпендикулярные оси: горизонтальная (ось X, абсцисс) и вертикальная (ось Y, ординат). Каждая точка задаётся парой чисел (x; y).

Расстояние между двумя точками A(x₁, y₁) и B(x₂, y₂): d = √((x₂−x₁)² + (y₂−y₁)²) — теорема Пифагора в координатах.
Середина отрезка AB: M = ((x₁+x₂)/2; (y₁+y₂)/2).

Прогрессии

Арифметическая прогрессия — каждый следующий член отличается от предыдущего на одно и то же число d (разность). Формулы: aₙ = a₁ + (n−1)·d; сумма S = n·(a₁ + aₙ)/2.

Геометрическая прогрессия — каждый следующий член умножается на одно и то же число q (знаменатель). Формулы: bₙ = b₁ · q^(n−1); сумма S = b₁·(qⁿ − 1)/(q − 1) при q ≠ 1.

aₙ = a₁ + (n−1)·d

bₙ = b₁ · qⁿ⁻¹

Округление чисел

Округление — замена числа на близкое, но «удобное» значение.

Правило: смотришь на следующую цифру после нужного разряда. Если она ≥ 5 — увеличиваешь последний оставляемый разряд на 1. Если < 5 — отбрасываешь остаток.
Пример: 3,746 → до десятых: смотрим на 4, это < 5, значит 3,7. До сотых: смотрим на 6, это ≥ 5, значит 3,75.
Разряды: …тысячи, сотни, десятки, единицы, . десятые, сотые, тысячные…

Масштаб

Масштаб — отношение размера на карте (чертеже) к реальному размеру. Записывается как 1 : N.

Например, масштаб 1 : 10 000 означает, что 1 см на карте = 10 000 см = 100 м в реальности.
Формулы: Реальное расстояние = расстояние на карте × N. Расстояние на карте = реальное / N.
Используется в географии, черчении, строительстве и архитектуре.

Реальное = карта × N

Карта = реальное / N
